What is Python?

Definition

Python is a high-level, interpreted programming language known for its simple syntax and readability. It emphasizes code readability and allows developers to express concepts in fewer lines of code.

History

Created by Guido van Rossum in 1991
Named after "Monty Python's Flying Circus"
Open source and free to use

Key Features

Easy to Read

Python's Main Characteristics

Interpreted

Python code is executed line by line at runtime, no compilation step required.

# Run directly without compiling
python script.py

Dynamic Typing

Variables can change types during runtime. No need to declare variable types.

x = 5 # int
x = "hello" # str

Object-Oriented

Supports classes, objects, inheritance, and encapsulation.

class Car:
def __init__(self):
self.speed = 0

Cross-Platform

Write once, run anywhere. Works on Windows, macOS, Linux, and more.

High-Level

Abstracts away complex details. Focus on problem-solving, not low-level operations.

# Simple file reading
with open('file.txt') as f:
content = f.read()

Rich Libraries

Vast ecosystem of libraries and frameworks for every need.

NumPyDjangoTensorFlow

Simple & Readable Syntax

Python Code

# Calculate factorial
def factorial(n):
if n <= 1:
return 1
return n * factorial(n-1)
print(factorial(5))

Other Languages (More Complex)

// Java example
public class Factorial {
public static int factorial(int n) {
if (n <= 1) return 1;
return n * factorial(n-1);
}
}

  • • Simple interface
  • • Interactive shell
  • • Basic debugging

Thonny

Python IDE for beginners

  • • Step-by-step execution
  • • Variable visualization
  • • Easy package management
  • • Perfect for learning

Online Editors

Repl.it, CodePen, etc.

  • • Instant sharing
  • • Cross-platform access
  • • Great for quick tests
